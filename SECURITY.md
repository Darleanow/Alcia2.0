# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| main    | yes       |

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Report them privately via [GitHub Security Advisories](../../security/advisories/new) or by email to the maintainer.

You will receive a response within 72 hours. If confirmed, a patch will be released as soon as possible.
